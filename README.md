# MCPanel - Minecraft Server Manager

![MCPanel](icon.png)

Современная панель управления для Minecraft серверов с поддержкой Paper, Velocity, Fabric, Forge и других ядер.

## 🚀 Возможности

- **Управление серверами** - Создание, запуск, остановка и удаление серверов
- **Поддержка ядер**:
  - Paper (рекомендуется)
  - Velocity 3.3.0 (прокси-сервер)
  - Vanilla
  - Fabric
  - Purpur
  - Forge
- **Консоль в реальном времени** - Просмотр логов и выполнение команд
- **Мониторинг** - CPU, RAM, игроки онлайн, аптайм
- **Управление плагинами** - Установка, удаление, редактирование конфигов
- **Система бэкапов** - Создание и восстановление резервных копий миров
- **Настройка конфигов** - Полная настройка server.properties и velocity.toml через UI
- **Velocity Forwarding** - Генерация и управление forwarding secret
- **Многоязычность** - Русский и английский интерфейс
- **Темная/светлая тема** - Настраиваемые цвета интерфейса

## 📥 Установка

### Windows

1. Скачайте последнюю версию из [Releases](../../releases)
2. Запустите `MCPanel Setup 1.0.0.exe`
3. Следуйте инструкциям установщика
4. Готово!

### Из исходников

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/mcpanel.git
cd mcpanel

# Установить зависимости
npm install

# Запустить в режиме разработки
npm start

# Собрать exe
npm run build
```

## 🎮 Быстрый старт

### Создание обычного сервера (Paper)

1. Нажмите **"+ Новый сервер"**
2. Выберите ядро **Paper** (рекомендуется)
3. Выберите версию Minecraft (например, 1.21.4)
4. Укажите порт (по умолчанию 25565)
5. Нажмите **"Создать сервер"**
6. Дождитесь загрузки ядра
7. Нажмите **"▶ Запустить"**

### Создание Velocity прокси

1. Нажмите **"+ Новый сервер"**
2. Выберите ядро **Velocity**
3. Версия будет автоматически 3.3.0
4. Укажите порт (например, 25565)
5. Нажмите **"Создать сервер"**
6. Откройте **Конфиг** → настройте серверы в JSON формате:
   ```json
   {
     "lobby": "127.0.0.1:25566",
     "survival": "127.0.0.1:25567"
   }
   ```
7. Настройте **Try Order**: `lobby,survival`
8. Если нужен forwarding:
   - Установите **Forwarding Mode** → **MODERN**
   - Нажмите **"🔄 Генерировать"** для создания секрета
   - Нажмите **"📋 Копировать"** и вставьте в `paper-global.yml` backend серверов
9. Нажмите **"Сохранить"**

## 🔧 Настройка Velocity Forwarding

### На Velocity (прокси):

1. Откройте конфиг Velocity
2. **Forwarding Mode** → **MODERN**
3. Нажмите **"🔄 Генерировать"** для создания секрета
4. Нажмите **"📋 Копировать"**
5. Сохраните конфиг

### На backend серверах (Paper/Spigot):

1. Откройте файл `config/paper-global.yml`
2. Найдите секцию `proxies.velocity`
3. Вставьте:
   ```yaml
   proxies:
     velocity:
       enabled: true
       online-mode: true
       secret: "ваш-скопированный-секрет"
   ```
4. Перезапустите серверы

## 📦 Управление плагинами

1. Откройте вкладку **"Плагины"**
2. Нажмите **"➕ Добавить плагин"**
3. Выберите `.jar` файлы плагинов
4. Плагины автоматически установятся в папку `plugins/`
5. Для редактирования конфигов:
   - Откройте папку плагина
   - Нажмите на `.yml` файл
   - Редактируйте и сохраните

## 💾 Бэкапы

### Создание бэкапа:

1. Откройте вкладку **"Бэкапы"**
2. Укажите название мира (по умолчанию `world`)
3. Нажмите **"💾 Создать бэкап"**
4. Бэкап сохранится в `AppData/Roaming/mcpanel/backups/`

### Восстановление:

1. Выберите бэкап из списка
2. Нажмите **"↻ Восстановить"**
3. Подтвердите действие
4. Мир будет восстановлен

## ⚙️ Настройки

Откройте **"⚙️ Настройки"** для настройки:

### Производительность:
- Максимум логов в памяти
- Интервал обновления статистики
- Время кэширования данных

### Интерфейс:
- Язык (Русский/English)
- Тема (Тёмная/Светлая)
- Цвета акцента

## 🛠️ Технологии

- **Electron** - Кроссплатформенный фреймворк
- **Node.js** - Backend
- **Vanilla JS** - Frontend (без фреймворков)
- **SQL.js** - База данных серверов
- **Adoptium Temurin** - Встроенная Java 8/17/21

## 📝 Системные требования

- **ОС**: Windows 10/11 (64-bit)
- **RAM**: Минимум 4 GB (рекомендуется 8+ GB)
- **Место на диске**: 500 MB для приложения + место для серверов
- **Java**: Автоматически загружается при необходимости
