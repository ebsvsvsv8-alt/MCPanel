# Contributing to MCPanel

Спасибо за интерес к проекту! Вот как ты можешь помочь:

## Как внести вклад

### Сообщить о баге
1. Проверь, что баг еще не был сообщен в Issues
2. Создай новый Issue с подробным описанием:
   - Шаги для воспроизведения
   - Ожидаемое поведение
   - Фактическое поведение
   - Скриншоты (если применимо)
   - Версия ОС и Node.js

### Предложить функцию
1. Создай Issue с тегом "enhancement"
2. Опиши, что ты хочешь добавить и зачем
3. Дождись обсуждения перед началом работы

### Pull Request
1. Fork репозиторий
2. Создай ветку: `git checkout -b feature/amazing-feature`
3. Внеси изменения
4. Проверь код: `npm run lint` (если настроен)
5. Протестируй: `npm run dev`
6. Commit: `git commit -m 'Add amazing feature'`
7. Push: `git push origin feature/amazing-feature`
8. Открой Pull Request

## Стиль кода

- Используй 2 пробела для отступов
- Комментируй сложную логику
- Называй переменные понятно (camelCase)
- Функции должны делать одну вещь

## Структура коммитов

```
feat: добавить поддержку BungeeCord
fix: исправить утечку памяти в processManager
docs: обновить README
style: форматирование кода
refactor: упростить логику загрузки Java
test: добавить тесты для backupManager
```

## Тестирование

Перед отправкой PR убедись что:
- [ ] Приложение запускается без ошибок
- [ ] Все существующие функции работают
- [ ] Новый код не ломает старый
- [ ] Нет console.error в продакшене

## Вопросы?

Открой Discussion или напиши в Issues!
