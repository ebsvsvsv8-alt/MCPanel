# Changelog

## [1.0.0] - 2026-04-17

### Added
- 🎉 Initial release
- ✨ Support for Paper, Vanilla, Fabric, Purpur, Forge, Velocity
- ⚡ Automatic Java 8, 17, 21 installation
- 📊 Real-time monitoring (RAM, CPU, players, uptime)
- 🎮 Built-in console with command execution
- ⚙️ Config editor for server.properties
- 🔄 Server management (start/stop/restart)
- 📁 Quick folder access
- 💾 Backup system for worlds
- 🔌 Plugin manager with Modrinth integration
- 📈 Performance monitoring and history
- 🛠️ Utility functions (port validation, disk space, etc.)
- 🎨 Modern dark UI with custom titlebar
- 🐛 Debug panel for development

### Features
- Optimized JVM flags (Aikar's flags)
- Graceful shutdown on app exit
- Log persistence between sessions
- Port availability checking
- Automatic version detection
- Multi-server support
- Cross-platform (Windows, macOS, Linux)

### Technical
- Built with Electron 29
- SQLite database (sql.js)
- Node.js backend
- Vanilla JS frontend
- No external dependencies for UI
